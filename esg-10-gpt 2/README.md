# Email Signature Generator + Case Converter

This repo contains:
- **Email signature generator** (HTML + rich copy)
- **Case converter** (upper/lower/title/sentence/toggle/alternating + camel/pascal/snake/kebab/dot/space)

Switch between tools using the top-right tabs (**Signature / Case**). The case converter supports **EN/RU**.

## Run locally

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
npm run preview
```
